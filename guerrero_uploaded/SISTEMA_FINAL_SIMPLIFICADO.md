## 📊 **SISTEMA FINAL - 3 HERRAMIENTAS**

### **1️⃣ OFERTAS** 🎁
**Para:** Marketing y promociones públicas
**Dónde:** Se muestran en landing page
**Ejemplo:** "20% OFF en inscripciones este mes"

**Cómo usar:**
```
Super Admin → Ofertas → + Nueva Oferta
- Título: "Verano 2024 - 20% OFF"
- Tipo: Porcentaje
- Valor: 20
- ✅ Mostrar en Landing
- Fechas de vigencia
```

---

### **2️⃣ CUOTAS PERSONALIZADAS** 💵 ⭐ **PARA DESCUENTOS A JUGADORES**
**Para:** Dar descuentos a jugadores específicos
**Dónde:** Super Admin → Cuotas → Cuotas Personalizadas
**Ejemplo:** "Juan Pérez paga RD$ 2,800 en vez de RD$ 3,500"

**Cómo usar:**
```
Super Admin → Cuotas → Cuotas Personalizadas → + Nueva Cuota

1. Buscar: "Juan Pérez"
2. Cuota: 2800
3. Razón: "Descuento 20% - Hermano menor"
4. Guardar

✅ Juan ahora paga RD$ 2,800 automáticamente
```

---

### **3️⃣ CUPONES** 🎟️
**Para:** Códigos que padres ingresan al pagar
**Dónde:** Super Admin → Cupones
**Ejemplo:** "Cupón VERANO2024 = 25% OFF"

**Cómo usar:**
```
Super Admin → Cupones → + Nuevo Cupón

- Código: VERANO2024
- Tipo: Porcentaje
- Valor: 25
- Límite: 50 usos
- Expira: 31/08/2024

✅ Los primeros 50 padres pueden usar este código
```

---

## 🎯 **¿CUÁNDO USAR CADA UNO?**

| Necesito | Uso |
|----------|-----|
| Promoción pública en landing | **Ofertas** |
| Descuento a Juan Pérez | **Cuotas Personalizadas** ⭐ |
| Descuento a 3 hermanos | **Cuotas Personalizadas** (una por cada uno) ⭐ |
| Código "HERMANOS20" para familias | **Cupones** |
| Beca completa para un jugador | **Cuotas Personalizadas** (cuota = 0) ⭐ |
| "20% OFF en verano" para todos | **Ofertas** |
