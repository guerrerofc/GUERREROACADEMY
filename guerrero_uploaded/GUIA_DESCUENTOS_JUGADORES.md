# 🎯 GUÍA PASO A PASO: ASIGNAR DESCUENTOS A JUGADORES
# Guerrero Academy

## 📋 **TU PREGUNTA:**
*"¿Cómo le agrego los descuentos a los jugadores específicos?"*

---

## ✅ **RESPUESTA RÁPIDA:**

Tienes **3 opciones**. La más recomendada es **Opción 2** (más fácil):

1. **Ofertas Asignables** - Para descuentos temporales con fechas
2. **Cuotas Personalizadas** - ⭐ **MÁS FÁCIL** - Para descuentos permanentes
3. **Cupones** - Para dar códigos a ciertos padres

---

## 🏆 **OPCIÓN RECOMENDADA: CUOTAS PERSONALIZADAS**

### ¿Por qué es la mejor?
- ✅ Más fácil y directa
- ✅ El descuento se aplica automáticamente cada mes
- ✅ No necesitas fechas ni configuraciones complicadas
- ✅ Puedes ver fácilmente quién tiene descuento

---

## 📝 **TUTORIAL COMPLETO: CUOTAS PERSONALIZADAS**

### **CASO PRÁCTICO:**
Tienes 3 hermanos: Juan, Pedro y María Pérez
- Cuota normal: RD$ 3,500
- Quieres darles 20% de descuento
- Nueva cuota: RD$ 2,800 (3,500 - 700)

---

### **PASO 1: IR A CUOTAS**

```
Super Admin → Menú lateral izquierdo → Cuotas 💵
```

---

### **PASO 2: IR A TAB "CUOTAS PERSONALIZADAS"**

Verás 2 tabs arriba:
- Cuotas por Categoría
- ⭐ **Cuotas Personalizadas** ← HAZ CLIC AQUÍ

---

### **PASO 3: CLIC EN "+ NUEVA CUOTA"**

Botón azul arriba a la derecha

---

### **PASO 4: BUSCAR PRIMER JUGADOR**

**Campo: "Buscar Jugador"**
- Escribe: `Juan`
- Aparecerán resultados
- Haz clic en: `Juan Pérez - 11-13 años`
- Se marcará con ✓

---

### **PASO 5: CONFIGURAR CUOTA**

**Cuota Mensual (RD$):**
```
2800
```

**Razón del Ajuste:**
```
Descuento 20% - Familia con 3 hermanos
```

**Fecha de Inicio:**
```
Hoy (o déjalo como viene)
```

**Fecha de Fin:**
```
DEJAR VACÍO (será indefinido)
```

---

### **PASO 6: GUARDAR**

Haz clic en **"Guardar"**

✅ **¡Listo!** Juan ahora tiene RD$ 2,800 como cuota

---

### **PASO 7: REPETIR PARA PEDRO Y MARÍA**

1. Clic en "+ Nueva Cuota"
2. Busca "Pedro Pérez"
3. Cuota: 2800
4. Razón: "Descuento 20% - Familia con 3 hermanos"
5. Guardar

Repite para María.

---

## ✅ **RESULTADO FINAL**

En **Super Admin → Cuotas → Cuotas Personalizadas** verás:

```
┌────────────────┬──────────┬─────────────────┬───────────┐
│ Jugador        │ Cuota    │ Razón           │ Acciones  │
├────────────────┼──────────┼─────────────────┼───────────┤
│ Juan Pérez     │ RD$ 2,800│ Descuento 20%   │ Desactivar│
│ Pedro Pérez    │ RD$ 2,800│ Descuento 20%   │ Desactivar│
│ María Pérez    │ RD$ 2,800│ Descuento 20%   │ Desactivar│
└────────────────┴──────────┴─────────────────┴───────────┘
```

---

## 🎯 **¿CÓMO FUNCIONA AHORA?**

### **Antes:**
- Juan pagaba: RD$ 3,500
- Pedro pagaba: RD$ 3,500
- María pagaba: RD$ 3,500
- **TOTAL:** RD$ 10,500

### **Ahora (con cuota personalizada):**
- Juan paga: RD$ 2,800 ✅
- Pedro paga: RD$ 2,800 ✅
- María paga: RD$ 2,800 ✅
- **TOTAL:** RD$ 8,400 (Ahorro: RD$ 2,100)

---

## 🔄 **¿Y SI QUIERO QUITARLE EL DESCUENTO?**

1. Ve a **Super Admin → Cuotas → Cuotas Personalizadas**
2. Busca el jugador
3. Haz clic en **"Desactivar"**
4. ✅ El jugador volverá a la cuota de su categoría (RD$ 3,500)

## 🆚 **COMPARACIÓN: CUOTAS vs OFERTAS vs CUPONES**

| Característica | Cuotas Personalizadas ⭐ | Ofertas Asignables | Cupones |
|----------------|-------------------------|-------------------|---------|
| **Facilidad** | ⭐⭐⭐⭐⭐ Muy fácil | ⭐⭐⭐ Media | ⭐⭐⭐⭐ Fácil |
| **Permanencia** | Indefinida o con fecha | Con fechas | Con fechas |
| **Aplicación** | Automática | Automática | Manual (padre ingresa código) |
| **Mejor para** | Hermanos, becas | Promociones temporales | Códigos promocionales |
| **Ejemplo** | "RD$ 2,800 siempre" | "20% OFF hasta dic" | "Usa VERANO2024" |

## ❓ **PREGUNTAS FRECUENTES**

### **P: ¿Puedo tener cuota personalizada Y oferta asignada?**
R: Sí, el sistema aplicará ambas.

### **P: ¿El padre ve el descuento?**
R: Sí, cuando vaya a pagar verá su cuota actual.

### **P: ¿Puedo cambiar la cuota después?**
R: Sí, edita o desactiva cuando quieras.

### **P: ¿Hay límite de jugadores con cuota personalizada?**
R: No, puedes tener cuantos quieras.
